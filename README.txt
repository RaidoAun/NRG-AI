Programmide jooksutamine:
Tuleb avada vastav .html fail, kui fail ei avane või on vigu tuleb veenduda,
et kasutatakse Google Chrome veebibrauserit antud failide avamiseks.
Kui eelnev ei õnnestu võib proovida avada Google Chrome ja aadressireale
ise sisestada faili asukoht ja sealt fail avada. Näiteks
C:\Users\Kasutaja\Desktop\flappy.html
Tuleb ka veenduda, et .html failidega kaasas on ka kaustad, kus on .js failid.
On oluline, et need kaustad .js failidega asuks samas kaustas, kus on .html failid.

Failide sisu:
Iga .html faili kohta on tehtud vastav kaust, kus on sellele vastav JavaScript kood
.js lõpuga failides. Tehisnärvivõrguga seotud kood on kaustas lib, seal on kolm faili:
NeuralNetwork.js, mis sisaldab tehisnärvivõrgu programmi, matrixmath.js, mis sisaldab
tehisnärvivõrgu programmi poolt kasutatavaid tehteid, evolution.js, milles on geneetilise
algoritmiga seotud kood. styles.css muudab .html failide kujundust.

.js failide lugemine:
Failide avamiseks tuleb teha parem klikk ja sealt alt valida programm,
millega soovitakse avada. Sobib ka Notepad, aga Notepad++ on parem. Töö koostamisel kasutati
Visual Studio Code programmi, sellega on võimalik avada terve töö kaust ja pääseda
kõikidele failidele korraga ligi.

.html failide lugemine:
Sobib ka avada veebibrauseriga ja vajutada parem klikk ja inspekteeri või
CTRL + SHIFT + i. Muudatuste tegemiseks oleks sobivam avada programmidega notepad++
või Visual Studio Code.

Mõndadel näidetel, näiteks XOR.html väljastatakse infot ka konsooli, mis avaneb kui vajutada
CTRL + SHIFT + j
